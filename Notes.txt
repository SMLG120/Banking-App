    Plaid Banking Functionality Handshake:
        1. Create a link token
        2. Pass generated link token to Plaid link
        3. Trigger flow of connecting bank account to application through Plaid link
        4. On Success, Plaid Link will provide temporary public token
        5. Exchange public token with permanent access token
        6. Exchange access token to get bank account information
        7. Processor:
            a. Generate a processor token by exchanging access token and bank account id to securely exchnage data between Plaid and the processor.
            b. Create a Funding Srouce using Dwolla.

        Dwolla Transfer:

            2. Pass receiver funding source url
